1. Health Data Analysis: Exercise Metrics

View Project on Colab

This project involves analyzing a one-month dataset of exercise data, including columns such as Duration, Date, Pulse, Max Pulse, and Calories burned. The dataset required comprehensive cleaning to address various inconsistencies and errors.

Using tools such as Pandas for data manipulation and Matplotlib and Seaborn for visualization, I transformed the raw data into insightful visualizations. The project includes the following analyses:

  • Trends in Heart Rate Over Time
  • Exercise Duration Analysis
  • Caloric Burn Distribution
  • Correlation Between Exercise Metrics

This project demonstrates my ability to clean and analyze health-related data, providing valuable insights into exercise patterns and health metrics. Explore the analysis and results to gain a clearer understanding of exercise habits and metrics.

3. Diwali Sales Data Analysis

View Project on Colab

In this project, I analyzed a substantial dataset related to Diwali sales, which includes 11,231 rows and 13 columns. The dataset required extensive cleaning and preprocessing to ensure accuracy and usability. My approach involved a detailed exploration of the data to uncover meaningful patterns and trends.

Using tools such as NumPy, Pandas, Matplotlib, and Seaborn, I created several visualizations to illustrate different aspects of the sales data. The visual analyses include:

  • Sales by Age Group
  • Category by Order
  • Orders by State
  • Marital Status by Category
  • Sales by Gender
  • Sales by Zone

This project demonstrates my proficiency in handling large datasets, performing data cleaning, and generating visual insights. The visualizations provide a comprehensive view of sales trends and customer demographics. Explore the analysis and results to see the detailed findings from the Diwali sales data.
